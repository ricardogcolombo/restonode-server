# restonode-server

You should have installed
  - node
  - yarn
  - MongoDB

### Installation
```
This is going to run in localhost:3000, make sure that your port is not used by another apps.
```

You should have running mongodb before start.

To install and start server just run
```sh
$ yarn install 
$ yarn start
```

### Test
```sh
$ yarn test 
```
